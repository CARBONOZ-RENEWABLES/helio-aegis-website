## Responsive Breakpoints

### Mobile (<768px)
- Single column
- Font sizes: 14px questions, 13px answers
- Minimum tap target: 48px height
- Horizontal scroll tabs
- Full-width items with padding

### Tablet (768–1023px)
- Single column
- Font sizes: 15px questions, 14px answers
- Tabs may wrap to 2 rows
- Adjusted spacing

### Desktop (1024px+)
- Single column
- Font sizes: 16px questions, 14px answers
- Tabs wrap naturally
- Max width: 860px centered
- Optimal reading width

## Animation Timings

```
Accordion Open/Close:
├─ Duration: 350ms
├─ Easing: cubic-bezier(0.16, 1, 0.3, 1)
└─ Properties: height, opacity

Chevron Rotation:
├─ Duration: 200ms
├─ Easing: ease
└─ Rotation: 0° → 180°

Hover Effects:
├─ Duration: 200ms
├─ Easing: ease
└─ Properties: border-color, background-color

Tab Transitions:
├─ Duration: 250ms
├─ Easing: ease
└─ Properties: color, border-color
```
